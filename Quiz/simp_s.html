What is Simplification?


Simplification means to find out a final answer for complex calculation


Simplification questions are asked to check the ability of a student to deal with numbers which can be in one of the following types:


  • Sometimes, a calculation is given and one of the numbers is missing from the calculation.To find out the missing number , we have to approximate the given numbers or do the basic operations.
  • Sometimes all the numbers are given with some operation between them and we have to simplify the calculations.

Rules related to simplification


  • Rule 1: Replace 'of' by multiplication & '/' by division.
    Example: Find 1/4 of 20.
    Solution (1/4)*20= 20/4 = 5
  • Rule 2: Always keep in mind the "BODMAS" rule.These operations have priorities in the same order as mentioned.
    Whenever we have more than one operation in the given calculation, we have to do the operations according to the priority specified by 'BODMAS'
    B- Bracket
    O-Of(means multiplication)
    D-Division
    M-Multiplication
    A-Addition
    S-Subtraction
    Example: Simplify (2+5) of 80
    Solution: Replace 'of' by multiplication= (2+5)*80
    Now we have three things: Bracket ,Addition, Multiplication . Acc to BODMAS we have to solve bracket first and then multiplication.
    (2+5)*80= 7*80= 560
  • Rule 3: Multiplication and division have the same priority(Do that operation first which is on left.)
  • Rule 4: Addition and subtraction have the same priority(Do that operation first which is on left.)
  • Rule 5: Don't hesitate in rounding the numbers to nearest integers.
  • Rule 6: Modulus of a Real Number: Modulus of a real number a is defined as
    a|= a, if a>0 or
    |a|= -a, if a<0
    Thus, |5|= 5 and |-5|= -(-5)=5
  • Rule 7: Virnaculum (or Bar): When an expression contains Virnaculam, before applying the 'BODMAS' rule, we simplify the expression under the Virnaculum.

Formulae Imp


  • (a - b)^2 = (a^2 + b^2 - 2ab)
  • (a + b)^2 = (a^2 + b^2 + 2ab)
  • (a + b) (a – b) = (a^2 – b^2 )
  • (a^3 + b^3) = (a + b) (a^2 – ab + b^2)
  • (a^3 - b^3) = (a - b) (a^2 – ab + b^2)
  • (a + b + c)^2 = a^2 + b^2 + c^2 + 2(ab + bc + ca)
  • (a^3 + b^3 + c^3 – 3abc) = (a + b + c) (a^r>2 + b^2 + c^2 – ab – bc – ac)
